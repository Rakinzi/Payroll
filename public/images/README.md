# Login Cover Image

Add a cover image for the login page:

**Filename:** `login-cover.jpg`
**Recommended size:** 1920x1080px or larger
**Format:** JPG, PNG, or WebP

The image will be displayed on the right side of the login page on desktop screens (hidden on mobile).

## Suggestions

- Use a professional business or payroll-themed image
- Ensure the image has good contrast for dark mode (it will be darkened and desaturated automatically)
- Avoid images with important text as they may be hard to read

## Fallback

If no image is added, the login page will show a muted gray background on the right side.
